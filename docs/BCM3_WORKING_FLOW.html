ğŸ“‹ Overview

This program is a Body Control Module (BCM) that:

  • Listens for CAN messages on vcan0 interface
  • Controls 3 GPIO outputs: Left Indicator, Right Indicator, Headlight
  • Blinks indicators when active (500ms ON, 500ms OFF cycle)
